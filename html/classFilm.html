Detailed Description

This class is defined in the Film.h header file and inherits from the Video class.

It adds two data members to the Video class, which are the number of chapters and an array of integers that represent the duration of each chapter. It also defines a setDurations function to set the values of the chapters array. It overrides the viewVariables and playMedia functions from the Base and Video classes.
