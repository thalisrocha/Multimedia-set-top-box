\doxysubsection{Detailed Description}
This class is defined in the \mbox{\hyperlink{Film_8h}{Film.\+h}} header file and inherits from the \mbox{\hyperlink{classVideo}{Video}} class. 

It adds two data members to the \mbox{\hyperlink{classVideo}{Video}} class, which are the number of chapters and an array of integers that represent the duration of each chapter. It also defines a set\+Durations function to set the values of the chapters array. It overrides the view\+Variables and play\+Media functions from the \mbox{\hyperlink{classBase}{Base}} and \mbox{\hyperlink{classVideo}{Video}} classes.
